Les Appels d'Asterisk

Le module vous permet de collecter plus d'informations sur les appels entrants et sortans de votre entreprise, et en details (appelant , appelé, date , durée ...)

toutes les informations sont sauvgardées dans une base de données, pour garder trace de chaque appel.

le module posséde une vue graphe qui permet d'avoir des statistiques de vos appels. .
